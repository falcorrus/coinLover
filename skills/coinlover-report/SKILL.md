# CoinLover Report

Генерация структурированного Markdown-отчета по финансам CoinLover.

## Workflow

1. **Определи период**: 
   - Если месяц не указан, используй текущий месяц и год.
   - Если указан (например, "за март"), переведи название месяца в число (1-12).
2. **Получи данные**: Выполни скрипт `scripts/generate_report.py` с аргументами `<year>` и `<month>`.
3. **Сохрани отчет**: 
   - Путь сохранения: `/Users/eugene/MyProjects/CoinLover/Obsidian_Docs/Reports/CoinLover_Expenses_Report_{YYYY}-{MM}-{DD}.md`.
   - Используй инструмент `write_file` для создания файла.
4. **Уведоми пользователя**: Подтверди успешное создание и выведи краткую сводку (Итого расходов).

## Пример вызова скрипта
```bash
python3 scripts/generate_report.py 2026 3
```
