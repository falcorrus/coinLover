# План: Базовая валюта (USD/EUR) при первом подключении

## Объектив
Реализовать возможность выбора одной из двух базовых валют (USD или EUR) в момент первого подключения пользователя к Google Таблице. Выбор фиксируется «навсегда» (один раз) и определяет все расчеты и символы в приложении.

## Изменения в Google Таблице (Source of Truth)
- **Лист `Metadata` (или `Settings`):**
    - Добавить ячейку (например, `A1`) для хранения значения `base_currency`.
    - Это значение будет запрашиваться при каждом `pull` или первом `connect`.

## Изменения в коде

### 1. `src/services/RatesService.ts`
- **Динамический Fetch:** Изменить URL API: `https://open.er-api.com/v6/latest/${baseCurrency}`.
- **Локальное хранение:** Добавить `APP_SETTINGS.STORAGE_KEYS.BASE_CURRENCY` для кэширования выбора в `localStorage`.
- **Метод `convertToBase`:** Сделать его универсальным (вместо привязки к USD).

### 2. `src/hooks/useFinance.ts`
- При инициализации (`useEffect`) проверять наличие `base_currency` в данных из таблицы.
- Если данные отсутствуют, инициировать состояние `needsBaseCurrencySelection: true`.

### 3. `src/App.tsx` (Onboarding Flow)
- **Модальное окно выбора:** Создать разовый UI (диалог), который появляется только если `ssId` задан, но `baseCurrency` еще не выбрана.
- **UI Рендеринг:** 
    - Заменить хардкод символа `$` на динамический `getCurrencySymbol(baseCurrency)`.
    - Все переменные типа `totalSpentUSD` переименовать или переосмыслить как `totalSpentBase`.

### 4. `src/services/googleSheets.ts`
- Добавить методы `getBaseCurrency(sheetId)` и `setBaseCurrency(sheetId, currency)`.

## Логика "Одного раза"
1. Пользователь вводит `ssId` или переходит по ссылке.
2. Приложение делает `pull` метаданных.
3. Если `base_currency` в таблице пуста:
    - Показываем окно: "Выберите валюту для учета (USD или EUR). Это действие нельзя будет изменить позже."
    - Пользователь выбирает.
    - Отправляем выбор в таблицу (фиксация).
    - Приложение перезагружается с новыми настройками.
4. Если `base_currency` уже есть — просто работаем в ней.

## Верификация
- При первом входе появляется выбор валюты.
- После выбора все суммы (баланс, траты, доходы) отображаются в выбранной валюте ($, €).
- API курсов валют запрашивает данные относительно выбранной базы.
- На другом устройстве при вводе того же `ssId` валюта подтягивается автоматически (без повторного выбора).
