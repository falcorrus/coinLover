# Дизайн-документ: Анимация транзакции CoinLover (Bybit -> Магазины)

## Обзор
Вертикальное (9:16) реалистичное видео для демонстрации простоты создания транзакции через Drag-and-Drop и автоматическую конвертацию валют.

## Сценарий (Таймлайн)
*   **0.0с - 1.5с:** Главный экран (Midnight Theme). Акцент на кошельке **Bybit** ($) и категории **Магазины**.
*   **1.5с - 3.0с:** Плавный Drag-and-Drop: иконка Bybit переносится на Магазины. Визуальный отклик (пульсация) при наведении.
*   **3.0с - 4.0с:** Выезд Numpad (Slide-in). Размытие заднего фона.
*   **4.0с - 6.0с:** Набор суммы "50" в поле USD. Поле "КУДА" мгновенно отображает "263" BRL.
*   **6.0с - 7.5с:** Нажатие на кнопку **СЕГОДНЯ** (ярко-зеленый акцент).
*   **7.5с - 8.5с:** Возврат на главный экран. Обновление суммы под иконкой Магазины.

## Технический стек
*   **Remotion:** Программная генерация видео на React.
*   **Tailwind/CSS:** Использование существующих стилей проекта (`index.css`).
*   **Lucide React:** Иконки (Wallet, ShoppingBag, и т.д.).
*   **Spring Animations:** Для реалистичного движения и инерции элементов.

## Архитектура компонентов Remotion
1.  `MainScreen.tsx`: Репликация главного экрана.
2.  `NumpadScreen.tsx`: Репликация экрана ввода суммы.
3.  `Cursor.tsx`: Визуализация касания/перетаскивания.
4.  `Composition.tsx`: Оркестрация кадров и переходов.

## Данные для демонстрации
*   **Источник:** Bybit (USD, фиолетовый).
*   **Цель:** Магазины (BRL, синий/оранжевый).
*   **Сумма:** 50 USD -> 263 BRL.
