# 📚 CoinLover: Решения и Устранение проблем

> Это краткий индекс основных исправлений. См. `!Docs/!SOLUTIONS.md` для полных деталей.

## 🔧 Платформенные исправления
- **Персистентность iOS PWA:** Используйте роутинг через пути (`/s/[ID]`) и Cookie на 1 год для связи Safari и PWA-контейнера.
- **Android Notch ("Челка"):** Используйте `env(safe-area-inset-top)` в обертке (wrapper), а не в фоне, чтобы сохранить консистентность цветов под челкой.
- **Кнопка "Назад" на Android:** Слушайте событие `backButton` через плагин Capacitor App, чтобы закрывать модалки вместо выхода из приложения.

## 🐛 Исправления багов
- **Смещение DND:** Перенесите `DragOverlay` в корень и удалите `transition: all` у перетаскиваемых элементов для устранения задержки курсора.
- **Отрицательные балансы:** Используйте робастный `parseNum` для обработки типографских минусов (`–`, `—`) из Google Таблиц.
- **Мерцание Сторис:** Всегда держите список сторис в DOM; рендерите плеер как параллельный фиксированный оверлей.
- **Логика Numpad:** Разделите поля Source/Target с функцией "Unlink" для ручного ввода курса обмена.
