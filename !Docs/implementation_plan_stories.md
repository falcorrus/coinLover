# Внедрение интерактивных Stories в CoinLover

Внедрение горизонтальной полосы Stories ("кружочков") в стиле Instagram/Telegram над списком кошельков. Фича повышает удержание, знакомит пользователя с возможностями приложения и предоставляет быструю живую аналитику с курсами валют в премиальном Linear дизайне (Glassmorphism, неоновые обводки, плавные микроанимации).

## User Review Required

> [!IMPORTANT]
> **Концепция Stories и источники данных:**
> Истории будут полностью динамическими и адаптивными:
> 1. **🚀 Обзор (Overview):** Живой подсчет баланса и трат за текущий месяц на основе реальных данных из `accounts` и `currentMonthTransactions`.
> 2. **💸 Финансовый дзен (Evening Ritual / Streak):** Динамический расчет "дня без трат" (No-Spend Day) с красивым анимированным огнем, если сегодня не было расходов, или мягкий призыв записать транзакции.
> 3. **📈 Курсы валют (Rates):** Котировки популярных валют (USD, EUR, BTC, USDT) с расчетом динамики изменения.
> 4. **💡 Про-советы (Hidden Gems):** Обучение жестам приложения (сортировка кошельков зажатием, Telegram-бот).
> 5. **⚡ Быстрые действия (Interactive Actions):** Слайд с интерактивными кнопками внутри сторис (мгновенная смена темы, быстрый переход к отчетам, открытие бота).

> [!TIP]
> **Сохранение состояния (Persistence):**
> Просмотренные истории будут сохраняться в `localStorage` по их уникальному ID, при этом обводка кружка будет становиться тусклой (как в Instagram/Telegram).

## Proposed Changes

Мы разделим изменения на два ключевых файла.

---

### [Component] Stories

#### [NEW] [StoriesSection.tsx](file:///Users/eugene/MyProjects/CoinLover/src/components/layout/StoriesSection.tsx)
Создание нового изолированного React-компонента, который рендерит:
1. **StoriesBar:** Горизонтальный скролл-список кружочков с градиентной обводкой для непросмотренных и тонкими серыми рамками для просмотренных.
2. **StoriesPlayer (Модальное окно):** Полноэкранный премиум-интерфейс в стиле мобильных приложений:
   - Сегментированный индикатор прогресса (Progress Bar) сверху с автоматическим перелистыванием (5 секунд на слайд).
   - Управление жестами: тап по правой части экрана — вперед, тап по левой — назад, долгое удержание (Long Press) — пауза.
   - Закрытие по крестику или свайпу вниз.
   - Уникальный адаптивный дизайн для каждого слайда со стеклянным фоном, цветным свечением и аккуратными шрифтами Outfit/Inter.

   - Проверка изменения статуса сторис на "просмотрено" (localStorage + тусклая обводка).
3. **Функциональный тест быстрых действий:** Нажатие на кнопку смены темы в Stories должно мгновенно менять тему всего приложения.
