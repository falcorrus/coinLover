# 🎨 CoinLover: UX и Дизайн-система

## 💎 Визуальная идентичность: Linear Style
- **Фон:** `#050505` (Чистый черный).
- **Акцент:** `#6d5dfc` (Яркий фиолетовый).
- **Типографика:** `Inter` (Интерфейс), `Space Grotesk` (Брендинг).
- **Поверхности:** Glassmorphism (`backdrop-blur-30px`), радиусы скругления 28px/14px.

## 🕹 Механика взаимодействия (DND)
- **Библиотека:** `@dnd-kit/core`.
- **Режимы:**
    - **Короткое нажатие:** История / Навигация.
    - **Долгое нажатие (0.6с):** Режим сортировки.
    - **Долгое нажатие (2.0с):** Модалка редактирования.
    - **Тянуть вниз:** Транзакция (Numpad).
- **Безопасность UI:** Слушатели событий привязаны *только* к иконке (кругу), позволяя использовать текст и фон для скролла.

## 🌑 Темы
1. **Black (Linear):** Премиальный глассморфизм с "парящими" кольцами в сторис.
2. **White:** Чистый, высококонтрастный минимализм.
3. **Mint (Neumorphism 2.0):** Мягкие тени, акценты Harvest Gold, тактильные ощущения.

## 📊 UX Аналитики
- **Динамический центр:** По умолчанию показывает "Всего", обновляется на "Название категории + Сумма" при наведении на сектор.
- **Иерархия:** Модель "Категория -> Тег". Общие области (Дом, Еда) — основные категории; детали (Названия магазинов) — Теги.
