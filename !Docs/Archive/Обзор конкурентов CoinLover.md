## О проекте CoinLover (coin.reloto.ru)

CoinLover — приложение для учета личных финансов с уникальным интерфейсом **Drag & Drop монеток** в стиле **Glassmorphism + Bento Grid**.

### Ключевые особенности

- 🪙 **Drag & Drop монетки** — перетаскивание иконок-монеток для создания расходов, доходов и переводов
- 🔄 **Умное различение жестов** — система автоматически отличает сортировку (боковое движение) от финансовых операций (вниз)
- ☁️ **Cloud Sync через Google Sheets** — двусторонняя синхронизация + разрешение конфликтов
- 🧮 **Встроенный калькулятор** с тегами и быстрым выбором даты
- 💱 **Мультивалютность** — поддержка нескольких валют с пересчетом в USD
- 🎨 **Premium-дизайн** — темная тема, Glassmorphism, Linear Style, плавные анимации
- 📊 **Аналитика** — помесячная статистика расходов по категориям

## 🥇 1. CoinKeeper — Главный аналог

> **Самый близкий конкурент по механике — перетаскивание монеток/кругов**

|              | Детали                         |
| ------------ | ------------------------------ |
| 🌍 Платформы | iOS, Android, Web              |
| 💰 Модель    | Freemium (подписка ~5 USD/мес) |
| 🔗 Сайт      | coinkeeper.me                  |
| ⭐ Рейтинг    | ~4.5 (App Store / Google Play) |

### За что хвалят CoinKeeper (Топ-5)

1. **🪙 Drag & Drop монетки с визуальным заполнением**
   Главная фишка — перетаскивание круглых иконок (доход - счет - категория). По мере расходования бюджета иконка категории визуально заполняется цветом — пользователь мгновенно видит, сколько бюджета осталось. Это самая близкая к CoinLover механика на рынке.
2. **📊 Наглядность бюджета**
   Круговые диаграммы и визуальный прогресс наполнения категорий расходов. Пользователь может в одном взгляде оценить финансовую картину месяца.
3. **👨‍👩‍👧 Семейный доступ**
   Возможность подключить партнера/семью к единому бюджету. Совместное ведение финансов — одна из самых востребованных функций.
4. **🏦 Подключение банков (CoinKeeper 3)**
   Авто-импорт транзакций из банков (но по отзывам — работает нестабильно для зарубежных банков).
5. **🎯 Геймификация процесса**
   Сам процесс перетаскивания монеток из дохода в расход превращает учет финансов в интерактивную игру, что значительно повышает мотивацию вести учет регулярно.

### Что критикуют

- Нестабильная синхронизация с банками
- Платные функции иногда не работают
- Подписочная модель раздражает (раньше было разовое)

## 🥈 2. Monefy

> **Минималистичный трекер с фокусом на скорость ввода**

|              | Детали                       |
| ------------ | ---------------------------- |
| 🌍 Платформы | iOS, Android                 |
| 💰 Модель    | Freemium (Pro ~3 USD разово) |
| 🔗 Сайт      | monefy.com                   |
| ⭐ Рейтинг    | ~4.6                         |

### За что хвалят (Топ-5)

1. **⚡ Самый быстрый ввод расходов**
   Добавление транзакции в 2 тапа — круговой интерфейс, где категории расположены вокруг центральной диаграммы. Нажал категорию - ввел сумму - готово. Многие пользователи называют это самым быстрым трекером.
2. **🍩 Круговая диаграмма в центре экрана**
   Главный экран — это большая наглядная круговая диаграмма расходов. Одного взгляда достаточно, чтобы понять, куда уходят деньги.
3. **🔒 Конфиденциальность (без банков)**
   Не подключается к банкам — все вводится вручную. Данные хранятся только на устройстве или в личном Google Drive / Dropbox. Пользователи ценят приватность.
4. **💱 Мультивалютность (150+ валют)**
   Удобно для путешественников и экспатов. Автоматический пересчет курсов.
5. **💸 Честная ценовая модель**
   Разовая покупка Pro-версии по низкой цене (нет подписки). Многие пользователи выделяют это как огромный плюс.

### Что критикуют

- Нет бюджетирования (только трекинг)
- Реклама в бесплатной версии
- Выглядит устаревшим, дизайн не обновлялся

### За что хвалят (Топ-5)

1. **🎨 Визуальная красота интерфейса**
   Один из самых красивых трекеров на рынке. Яркие графики, приятные цвета, интуитивно понятная навигация. Многие выбирают именно за внешний вид.
2. **👫 Общие кошельки для пар/семей**
   Реал-тайм синхронизация общих кошельков. Особенно популярно у пар для контроля общих расходов (квартира, продукты, путешествия).
3. **🏦 Авто-импорт из банков**
   Premium-пользователи могут подключить банки для автоматической загрузки и категоризации транзакций. ИИ учится на истории расходов.

   Автоматическая конвертация, удобно для номадов и международных пользователей.

   Цветные графики, тренды, аналитика расходов по периодам — без необходимости в Excel.

### Что критикуют

- Бесплатная версия очень ограничена (1 кошелек, только ручной ввод)
- Баги при синхронизации с банками
- Нет продвинутых аналитических инструментов

### За что хвалят (Топ-5)

1. **🤖 Лучшее авто-распознавание банковских уведомлений**
   Считает транзакции из SMS и push-уведомлений от банков. Автоматическая категоризация. Минимум ручного ввода. Поддержка 3300+ банков мира — больше, чем у большинства конкурентов.
2. **📋 Глубокое бюджетное планирование**
   Планирование будущих расходов, регулярных платежей, установка лимитов по категориям. Пользователи ценят возможность видеть будущие траты заранее.
3. **📈 Полноценная аналитика**
   Детальные графики, отчеты по динамике, сравнение месяцев. Все в одном месте — наличные, карты, вклады, инвестиции, криптовалюты.
4. **👨‍👩‍👧 Совместное ведение бюджета**
   Семейный бюджет с разграничением прав. Популярная функция для семей.
5. **🔌 API для разработчиков**
   Наличие публичного API — можно интегрировать с собственными системами. Ценится гиками и продвинутыми пользователями.

### Что критикуют

- Нестабильная синхронизация с банками (главная жалоба 2024-2025)
- Устаревший веб-интерфейс
- Высокая стоимость подписки для одного пользователя
- Сложный для новичков (перегруженный интерфейс)

2. **📅 Гибкие бюджетные периоды с переносом остатка**

   Одно из лучших решений для мультивалютного учета. Автоматическое обновление курсов. Поддержка криптовалют.
4. **🏦 14 000+ банков (Medici план)**
   Самое широкое покрытие по банковским подключениям. Подходит для пользователей в нестандартных юрисдикциях.
5. **📊 Детальные отчеты и экспорт**
   Разнообразные графики, возможность экспорта в Excel/CSV. Пользователи хвалят точность и полноту отчетов.

### Что критикуют

- Дизайн монстриков нравится не всем
- Ограниченная бесплатная версия

## 🎯 Главные выводы

### Уникальные преимущества CoinLover

1. **🪙 Единственный аналог с монетками + Glassmorphism**
   CoinKeeper — единственный конкурент с DnD-монетками, но его дизайн значительно более консервативный. CoinLover — единственный, кто сочетает drag-and-drop монеток с премиальным dark-mode Glassmorphism дизайном.
2. **🔧 Трехступенчатая логика DnD**
   Ни один конкурент не реализует разделение жестов: нажатие (scale 0.9) - захват/сортировка (scale 1.1) - покой (scale 1.0). Это уникальная техническая реализация.
3. **🔄 Conflict Resolution**
   Механизм обнаружения и разрешения конфликтов при многоустройственной работе — отсутствует у всех конкурентов (они либо «последний пишет», либо не поддерживают мульти-устройства).
4. **💰 Полностью бесплатное**
   Все конкуренты (кроме Monefy с разовой покупкой) требуют подписку. CoinLover — полностью бесплатный.

## 🏆 Итоговый вердикт

**CoinLover занимает уникальную нишу** на пересечении:

- ✅ Механики перетаскивания монеток (как CoinKeeper)
- ✅ Премиального дизайна (на уровне или лучше Spendee)
- ✅ Полной бесплатности (без аналогов)
- ✅ Уникальной синхронизации через Google Sheets

**Главный конкурент — CoinKeeper**, но CoinLover превосходит его в:

- Дизайне (Glassmorphism vs устаревший flat)
- Технической реализации DnD (трехступенчатая логика с dnd-kit vs базовый drag)
- Ценовой модели (бесплатно vs подписка)

**Что может усилить позиции**: визуальный индикатор бюджета на иконках категорий (ключевая причина, по которой пользователи предпочитают CoinKeeper).
