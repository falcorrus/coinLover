# 🗺 CoinLover: Планы развития (Roadmap)

## 🏢 Переход на Multi-Tenant
1. **Auth:** Интеграция Supabase/Firebase Auth.
2. **Изоляция:** Модель "Таблица на пользователя" (Bring Your Own Database).
3. **Онбординг:** Пошаговый процесс копирования шаблона и привязки SSID.
4. **Бэкенд:** Обновление `api/sheets.ts` для динамического приема `spreadsheetId` из Auth-токена.

## 🤖 AI Функции
- **In-App Analyst:** Архитектура для анализа паттернов трат через Gemini API.
- **Ввод на естественном языке:** "Потратил 500 на ужин с друзьями" -> авто-транзакция.
- **Персонализированные советы:** Динамические сторис на основе финансового поведения (Финансовая карма).

## 🌍 Экспансия
- **Локализация:** Продолжение L10n для английского, сербского и португальского языков.
- **Currency API:** Надежный провайдер курсов для авто-конвертации в Numpad.
