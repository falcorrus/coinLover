# Сравнение Светлых Тем (Trend 2026)

Этот отчет сравнивает вашу текущую светлую тему с ведущими дизайн-системами года.

## 1. Ваша Тема (Current)
*   **Стиль:** Glassmorphism + Bento Grid.
*   **Формы:** Круги (Wallets, Categories). Это создает дружелюбный, "игрушечный" вид.
*   **Стекло:** Высокая прозрачность в шапке и карточках.
*   **Акценты:** Мягкие цвета, тонкие иконки.

## 2. Apple (Liquid Glass - iOS 26)
*   **Ключевое отличие:** Рефракция (преломление) и динамические отражения. Стекло не просто моет фон, оно "играет" со светом.
*   **Формы:** Плавные скругления (Squircles), радиус 22-28px.
*   **Применение:** Используется для создания ощущения "премиальности" и глубины.

## 3. Linear (Linear Style)
*   **Ключевое отличие:** Строгая сетка и высокая контрастность.
*   **Светлая тема:** "Frosted Glass" (матовое стекло) с очень тонкими границами (1px #E2E2E3).
*   **Формы:** Прямоугольники с радиусом 8-12px. Круги почти не используются для контейнеров.
*   **Философия:** Фокус на данных и скорости.

## 4. Vercel (Geist)
*   **Ключевое отличие:** Цифровая точность, "Pixel Grid".
*   **Стекло:** Используется минимально, только для навигации. Основной упор на чистый белый и серые уровни.
*   **Формы:** Плотные скругления 8px.

## Рекомендации по улучшению
1.  **Радиусы:** Попробуйте заменить круги на "мягкие квадраты" (радиус 24-28px) для секций "Wallets" и "Categories". Это сделает интерфейс более современным (в стиле Apple/Linear) и позволит разместить больше информации внутри карточки.
2.  **Границы (Borders):** Добавьте едва заметную границу `1px solid rgba(255,255,255,0.08)` для стеклянных элементов в светлой теме — это создаст эффект "алмазной грани".
3.  **Рефракция:** В светлой теме вместо простого `blur` попробуйте добавить легкий белый градиент сверху вниз (от `white/40` до `white/10`), чтобы имитировать падение света.
